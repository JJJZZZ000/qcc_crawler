1. python环境搭建
2. 安装chrome
3. Company.txt填写需要爬取的公司名称，可以用","（中英符号都可）、换行、Tab区分
4. XPath填写需要的信息，用":"（中英符号都可）分隔。
例如： 名称：text：/html/h1
名称是自定义姓名， text是HTML节点的属性类型（例如href是链接等等，一般眼睛看到的都是text），
/html/h1 是信息所在节点的xpath。可以通过浏览器F12开发者模式进行选取，要选择的足够精确。
不同的属性需要换行分隔
详情请看XPath.txt文件的用例
5. 运行init.py，文件功能支持自动更新chromedriver
6. 在Result文件夹中自动生成了结果文件，请查收

注意：
1. 文件名不要随便改
2. 不要挂VPN
3. 程序运行后需要手都扫码登录一次
